Максимальное количество принятых сообщений за один запрос. The maximum number of messages to receive per request. Amazon SQS не вернёт больше сообщений за запрос, но быть может вернёт меньше. Значения могут быть от 1 до 10. По умолчанию стоит 10.

Нет гарантии что ВСЕ сообщения обязательно вернуться.

Более высокие значения сокращают количество запросов, которые нужно выслать. Рекомендуется использовать наивысшее значение. Уменьшайте его только если проблемы стабильности действительно решаются меньшими значениями.
